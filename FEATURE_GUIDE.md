- They'll work automatically when you reuse the template

## â“ FAQ

**Q: Where are templates stored?**
A: Templates are stored in your database, not in browser local storage. This means they're accessible from any device.

**Q: Can I edit a saved template?**
A: Yes! Select the template in a campaign, make your changes, and save it as a new template or update the existing one.

**Q: Are templates shared between stores?**
A: No, templates are specific to each store for security and organization.

**Q: How many templates can I save?**
A: There's no limit! Save as many as you need.

**Q: Can I export/import templates?**
A: Not yet, but this feature is planned for a future update.

**Q: What happens if I delete a template?**
A: The template is permanently removed from your saved templates. Campaigns that already used this template are not affected.

## ğŸ†˜ Troubleshooting

**Problem:** "Save as Template" button is disabled
- **Solution:** Make sure you've entered some content in the message/email

**Problem:** My saved templates don't appear
- **Solution:** Check that you're on the "My Templates" tab, not "Templates"

**Problem:** Can't delete a template
- **Solution:** Make sure you have permission to manage templates for this store

**Problem:** Template content looks different when loaded
- **Solution:** For email templates, some styling may need adjustment after loading
